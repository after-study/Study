# 6장 자바스크립트 객체

### 프로토타입 활용 상속

자바와 같은 대부분의 강 자료형 언어에서는 클래스의 메소드가 클래스와 동시에 정의되지만, 자바스크립트에서는 함수가 클래스의 자바스크립트 Object 속성으로 추가돼야 한다.

클래스 생성자를 통해서 호출

```javascript
function ExampleClass() {
  this.name = "Javascript";
  this.sayName = function () {
    conosle.log(this.name);
  };
}

//신규객체
var example1 = new ExampleClass();
example1.sayName(); //"Javascript"
```

프로토타입을 활용하여 상속

```javascript
function ExampleClass(){
    this.name = "Javascript";
    this.sayName = function(){
        conosle.log(this.name);
    }
}

//신규객체
var example1 = new ExampleClass();

ExampleClass.protytype.sayName() = functiom(){
    console.log(this.name)
}

example1.sayName(); //"Javascript"
```

위 클래스는 생성자에서 sayName 함수를 동적으로 추가한다. 이러한 패턴을 프로토타입 활용 상속이라고 한다.
위 방법은 자바스크립트의 유일한 상속 방법인데, .prototype 속성을 사용한 다음 함수의 이름을 지정하기만 하면 된다.
.protoype 속성을 사용해서 동적으로 확장할 수 있다.

### 생성자와 변수

자바스크립트에서 클래스의 변수가 해당 클래스 객체의 속성이기 때문에 this.propertyName으로 선언된 모든 속성을 공개적으로 접근할 수 있다.
이는 해당 객체의 속성들을 다른 범위에서도 직접 접근할 수 있다는 의미이다.
