# 자바스크립트 문자열

## 1️⃣ String 객체의 내장함수와 문자열 처리하는 방법을 알아보자!

### 문자열 접근 String.charAt

```javscript
'dog'.charAt(1) // "o"를 반환
```

0부터 시작하는 인덱스 위치에 있는 문자를 반환

### 문자열 비교

<문자열 하나일 경우>

```javascript
var a = 'a';
var b = 'b';

console.log(a < b) 'true'를 출력
```

각 문자의 유니코드값을 비교한다.

<다른 길이의 문자열일 경우>

```javascript
var a = "add";
var b = "b";
console.log(a < b); // 'true'를 출력합니다.
```

<첫 글자가 같은 경우>

```javascript
var a = "add";
var b = "ab";
console.log(a < b); // 'false'를 출력합니다.
```

첫 글자가 같으므로 두 번째 글자를 비교합니다. 'd'가 'b'보다 크므로 false이다.

### 문자열 검색 String.indexOf(searchValue[, fromIndex])

검색하고자 하는 문자열을 매개변수로 받고 일치하는 문자열을 찾지 못하면 -1을 반환한다.

```javascript
"Red Dragon".indexOf("Red"); // 0 반환
"Red Dragon".indexOf("RedScale"); // -1 반환
"Red Dragon".indexOf("Dragon", 0); //인덱스0부터 검색시작해서 4에서 Dragon의 시작위치이여서 4를 반환
"Red Dragon".indexOf("Dragon", 4); // 4로시작해서 4를 반환
"Red Dragon".indexOf("", 9); // 9를 반환 -> 빈문자열은 모든위치에 존재한다고 간주하여 9반환
```

### 문자열 분해 String.split

하나의 매개변수를 입력받아 부분 문자열 배열 생성

```javascript
var test1 = "chicken";
test1.split(""); // ['c','h','i' .... ];
```

### 문자열 바꾸기 String.replace

문자열 변수 내에 특정 문자열을 다른 문자열로 대체

```javascript
"Wizard of Oz".replace("Wizard", "Witch"); //Witch of Oz
```

## 2️⃣ 정규표현식

정규 표현식은 문자열 패턴을 정의하고 검색, 치환, 추출하는 강력한 도구며 복잡한 문자열 작업을 간결하게 처리할 수 있다

### RegExp의 두 가지 함수

- search(): 문자열 내에 일치하는 문자열을 찾습니다. 일치하는 문자열의 인덱스를 반환합니다.
- match(): 일치하는 문자열을 찾습니다. 모든 일치하는 문자열을 반환합니다.

자바스크립트 String 객체에는 정규 표현식과 관련된 다음 두 가지 함수가 있는데 두 함수는 RegExp 객체를 인자로 받습니다.

- exec(): 문자열 내에 일치하는 문자열을 찾습니다. 일치하는 첫 번째 문자열을 반환합니다.
- test(): 문자열 내에 일치하는 문자열을 찾습니다. true 또는 false를 반환합니다.

### 기본 정규 표현식 규칙

- ^: 문자열/줄의 시작을 나타냅니다.
- \d: 모든 숫자를 찾습니다.
- [abc]: 괄호 내의 모든 문자를 찾습니다.
- [^abc]: 괄호 내의 문자들을 제외한 모든 문자를 찾습니다.
- [0-9]: 괄호 내의 모든 숫자를 찾습니다.
- [^0-9]: 괄호 내의 숫자들을 제외한 모든 숫자를 찾습니다.
- (x|y): x 또는 y를 찾습니다.

### 3️⃣ 인코딩

인코딩은 컴퓨터 과학 분야에서 효율적인 전송 혹은 저장을 위해 문자들을 특수 포맷으로 표현하는 포괄적인 개념이다.
모든 컴퓨터 파일 유형은 특정 구조로 인정된다.

### Base64 인코딩

btoa() 함수는 문자열로부터 Base64 인코딩된 ASCII 문자열을 생성한다. 문자열의 각 문자는 바이트로 취급된다.
</br> atob 함수는 Base64 인코딩을 사용해 인코딩된 자료의 문자열을 디코딩한다.
